Introduction

STM8CubeMX is a graphical tool for 8-bit STM8 microcontrollers.

STM8CubeMX has the following key features:
  • Easy microcontroller selection covering whole STM8 portfolio.
  • Board selection from a list of STMicroelectronics boards.
  • Easy microcontroller configuration (pins, clock tree)
  • Easy switching to another microcontroller belonging to the same series by importing a previously-saved configuration to a new MCU project.
  • Generation of configuration reports.
  • Power consumption calculation for a user-defined application sequence.

    Installing STM8CubeMX

    To install STM8CubeMX, follow the steps below:
    • Extract (unzip) the whole content of the the latest STM8CubeMX installation package from http://www.st.com/stm8 into the same directory
    • on Windows
      • Make sure you have administrators rights
      • Double click on the SetupSTM8CubeMX-1.1.0.exe file
    • On Linux
      • Make sure you have access rights to the target installation directory, you can run the installation as root (or sudo)  to install STM8CubeMX on shared directories
      • Double click (or launch from the console window) on the SetupSTM8CubeMX-1.1.0.linux file
    • On MacOS
      • Make sure you have access rights to the target installation directory, you can run the installation as root (or sudo)  to install STM8CubeMX on shared directories
      • Double click (or launch from the console window) on the SetupSTM8CubeMX-1.1.0 application
    • After the installation you can safely remove the content of the zip from your disk.
    • Please refer to the STM8CubeMX user manual for more details on System requirements or other possible installations.

    Running STM8CubeMX

    • on Windows
      • Select STM8CubeMX from Program Files > ST Microelectronics > STM8CubeMX.
      • Or double-click STM8CubeMX icon on your desktop.
    • on Linux
      • launch the STM8CubeMX executable from the STM8CubeMX installation directory
    • on MacOS
      • launch the STM8CubeMX application from the launchpad

    Uninstalling STM8CubeMX

    To uninstall STM8CubeMX, follow the steps below:
    • on Windows
      • Open the Windows Control panel.
      • Select Programs and Features to display the list of programs installed on your computer.
      • Right click on STM8CubeMX and select the uninstall function.
    • on Linux
      • use a file explorer, go in the Uninstaller directory of the STM8CubeMX installation and double click on the startuninstall desktop shortcut
      • or launch manually the uninstallation with java -jar <install path>/Uninstaller/uninstaller.jar
    • on MacOS
      • use a file explorer, go in the Uninstaller directory of the STM8CubeMX installation and double click on the uninstaller.jar
      • or directly move the STM8CubeMX application in the recycle bin
